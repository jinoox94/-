2. 완전한 요청 템플릿 (Ver 2.0)

📋 복사해서 사용할 요청문

[기업명]을 다음 구조의 HTML 기업분석보고서로 작성해줘:

📄 문서 형태:
- HTML 웹페이지 (인쇄 가능)
- Tailwind CSS + Font Awesome 아이콘
- 반응형 디자인
- 전문적인 비즈니스 톤앤매너

🏗️ 필수 구조:
1. Header (기업명, ICP 스코어)
2. Executive Summary (핵심 결론, 기회/리스크)
3. 기업 기본정보 (매출, 임직원, 사업영역)
4. 5개 항목 상세분석
5. BANT 분석 (Budget/Authority/Need/Timing)
6. ICP 평가 상세 (점수 기준, Hot/Warm/Cold Lead 분류)
7. 최신 뉴스/보도자료 요약 (실제 링크 포함)
8. 미팅 전략 (대응방안, 예상질문)
9. 액션플랜 (즉시/단기/중장기)
10. 리뉴얼 전략 제안서 (새로 추가)

🎨 디자인 요구사항:
- 그라디언트 헤더 (blue-purple)
- 색상별 카드 구성 (Green/Blue/Yellow/Red/Purple)
- 아이콘 포함 섹션 제목
- 점수 표시 (배지 형태)
- 시각적 구분선 및 여백
- 호버 효과 및 애니메이션

📊 핵심 분석 요소:
- ICP 스코어링 (7개 항목별 점수)
- Hot Lead 기준 (50% 이상)
- BANT 4요소 상세 평가
- 영업 기회 분석
- 실행 가능한 액션플랜
- 맞춤형 리뉴얼 전략 (새로 추가)

🔗 보도자료 요구사항:
- 최신 뉴스 3-5개 수집
- 실제 접근 가능한 링크 포함
- 날짜 및 출처 명시
- 영업 기회와 연계 분석

💡 리뉴얼 전략 제안서:
- 이커머스 사이트: 고도몰/카페24 등 기반 전략
- 기업 사이트: M-CMS 활용 전략
- 예산 규모별 단계적 접근
- ROI 및 성과 지표 제시
- 사전 컨설팅/예산 확보 지원

4. 보도자료 링크 수집 방법

🔍 수집 전략

1차 검색 키워드

"[기업명] 2024 2025 보도자료 뉴스"
"[기업명] 최신 동향 사업 확장"
"[기업명] 수상 실적 발표"

2차 세부 검색

"[기업명] 디지털 전환 투자"
"[기업명] 브랜드 가치 상승"
"[기업명] 정기 대의원회 실적"

📋 링크 품질 기준

✅ 우수 소스

  • • 공식 보도자료 사이트
  • • 업계 전문 언론사
  • • 정부/협회 발표 자료
  • • 최근 6개월 이내 뉴스
  • • 접근 가능한 URL

❌ 제외 소스

  • • 개인 블로그/카페
  • • 접근 불가 링크
  • • 2년 이상 된 자료
  • • 광고성 기사
  • • 부정적 뉴스만

📝 링크 포함 템플릿

<div class="bg-green-50 border-l-4 border-green-500 p-4">
  <h4 class="font-semibold text-green-800">
    <a href="[실제_링크_URL]" target="_blank" class="hover:underline">
      [뉴스_제목]
    </a>
  </h4>
  <p class="text-green-700 text-sm">[뉴스_요약_내용]</p>
  <span class="text-xs text-green-600">[날짜] | [출처]</span>
</div>

5. 리뉴얼 전략 제안서 작성 가이드

이커머스 사이트 전략

🛒 플랫폼 분석

  • • 현재 솔루션 (고도몰/카페24/메이크샵 등)
  • • 한계점 및 개선 필요사항
  • • 경쟁사 벤치마킹
  • • 모바일 최적화 수준

💡 고도화 전략

  • • UX/UI 개선 방안
  • • 결제/배송 시스템 최적화
  • • 개인화 추천 시스템
  • • 데이터 분석 대시보드

📊 예상 효과

  • • 전환율 향상 (예: 15-30%)
  • • 평균 주문 금액 증가
  • • 고객 재구매율 개선
  • • 운영 효율성 증대

M-CMS 기업사이트 전략

🏢 기업 사이트 분석

  • • 현재 CMS 솔루션 현황
  • • 콘텐츠 관리 효율성
  • • SEO 최적화 수준
  • • 반응형 웹 대응도

⚙️ M-CMS 고도화

  • • 직관적 관리자 시스템
  • • 다국어/다사이트 지원
  • • API 연동 확장성
  • • 보안 및 백업 시스템

🎯 브랜드 강화

  • • 브랜드 아이덴티티 강화
  • • 사용자 경험 최적화
  • • 콘텐츠 마케팅 지원
  • • 리드 제너레이션 시스템

📋 제안서 표준 구조

1단계: 현황 분석

  • • 현재 시스템 진단
  • • 문제점 도출
  • • 경쟁사 분석
  • • 개선 우선순위

2단계: 솔루션 제안

  • • 기술적 솔루션
  • • 단계별 개발 계획
  • • 예산 및 일정
  • • 기대효과 및 ROI

3단계: 실행 계획

  • • 프로젝트 일정
  • • 팀 구성 및 역할
  • • 성과 측정 지표
  • • 사후 지원 방안

6. ICP 점수 체계 (변경 없음)

📊 7개 평가 항목 (총 100점)

1. 기업 규모 20점
2. 산업군 적합도 15점
3. 예산 타당성 15점
4. 디지털 니즈 15점
5. 의사결정 구조 15점
6. 협업 경험 10점
7. 장기 협업 10점

🎯 리드 분류 기준

Hot Lead (50점 이상)

🔥 우선 타겟 고객 - 전환율 70%

Warm Lead (30-50점)

❤️ 관심 고객 - 전환율 40%

Cold Lead (30점 미만)

❄️ 장기 육성 - 전환율 15%
